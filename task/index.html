  1. 创建一个 gitpage 博客
    • 知乎2020还在靠这种文章赚点击的Hexo+gitpage教程
  2. 打开 dash 查看相关的 Markdown 语法，下载 Markdown 编辑器
  3. 复习Java reflect 和 git 学习过的内容，将他们分别写成 Markdown 文章，并尝试发布到你的 gitpage。文章的内容你得加入你自己的想法，而不是直接指令或者代码一扔。你要写清楚功能，比如，3种reflect机制获取到的类对象分别适用于什么场景？这是一个问题，你要根据这种问题去开阔你的文章。写文章就是将发现的问题进行一个解决方案的梳理和归纳。
  4. 将 Java reflect 的作业完成，写成 Markdown 文章，梳理完成过程以及代码。
  5. 过一遍下面的技能树，明白一下每个技术点用到的技术的名字，大部分都是相互可替换的。但你要有印象，某个技术点是用来作什么的。
  6. 有时间到 how2j.cn 中过一遍 Java的基础，中级和高级。重点查看 泛形, 多线程,注解
